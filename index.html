ShellOS

The 13MB Handmade Linux Experiment

What is ShellOS?

ShellOS is not just another minimal distribution. It is a proof-of-concept operating system where the core userspace is not borrowed from GNU or BusyBox, but built by hand.

The goal was to understand the Linux kernel by building the bricks that sit on top of it. Everything from process initialization to file management is handled by custom C code, compiled statically for maximum portability and minimalism.

Handmade in C

While most distros package existing tools, ShellOS features a custom suite of utilities written specifically for this project. There is no glibc bloat here. The following tools were coded from scratch:

shell
init
ls
echo
cat
rm
rmdir
mkdir
clear
minifetch
poweroff
reboot

The Shell

The heart of ShellOS is its custom shell. Unlike Bash or Zsh, which are complex programs with thousands of features, the ShellOS shell focuses on the absolute essentials required to navigate a system.

It features a custom tokenizer for command parsing, built-in implementations of cd and exit, and direct path execution. It serves as the primary interface between the user and the kernel.
