ShellOS

13MB. Handmade. Pure C.

> What is ShellOS?

ShellOS is an experimental Linux distribution. It discards the standard GNU coreutils and BusyBox environments in favor of a fully handmade userspace.

There are no complex libraries, no package managers, and no bloat. Just the Linux Kernel and a collection of C binaries written from scratch to understand how an operating system thinks.

> The Handmade Toolset

Every command you type is a custom C program compiled statically. This ensures total independence from shared libraries.

/bin/sh
/bin/init
/bin/ls
/bin/echo
/bin/cat
/bin/rm
/bin/rmdir
/bin/mkdir
/bin/clear
/bin/poweroff
/bin/reboot
/bin/minifetch

> The Shell

The ShellOS shell is a minimalist command interpreter. It handles execution, path resolution, and argument parsing without the overhead of Bash.

It includes internal implementations for cd, and directly interfaces with the kernel to launch your processes.
